# Login System - RPG Game Suite

A comprehensive C++ application combining user authentication, book management, and an RPG game.

## Features
- **User Authentication** - Register, login, and profile management with IP tracking
- **Profile Management** - Edit username, age, password, and phone number
- **Admin Panel** - Delete users, view user list, reset all users
- **Book Management** - Full CRUD operations with CSV and formatted file support
- **RPG Game** - Choose from Warrior, Mage, or Rogue classes
- **Turn-Based Combat** - Battle enemies with skills and items
- **Mini-Games** - Guess the Number and Dice Game
- **Calculator** - Basic arithmetic operations
- **Data Persistence** - Automatic saving with file storage

## How to Build

### Windows (MinGW)
```
g++ -std=c++17 -o login.exe main.cpp
```
## Linux/Mac 
```
g++ -std=c++17 -o login main.cpp 
```
## How to Run
Windows
```
./login.exe 
```
Linux/Mac
```
./login 
```
