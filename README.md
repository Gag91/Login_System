# Login_System
A comprehensive C++ application combining user authentication, book management, and an RPG game.

## Features
- User registration and login with IP tracking (local IP, i do not have acces to any of the users info)
- Profile management (edit username, age, password, phone)
- Admin panel with user management
- Book management system (CRUD operations)
- RPG game with character classes (Warrior, Mage, Rogue)
- Turn-based combat system
- Mini-games (Guess the Number, Dice Game)
- Calculator
- Data persistence using file storage (#include <fstream>)

## How to Build
```
g++ -std=c++17 -o login.exe main.cpp ```

## How to Start
```
./login.exe ```
